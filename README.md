# pyaam - active appearance model

active appearance models implemented in python

## Instructions

Download MUCT dataset:

    python -m pyaam.muct

View MUCT dataset:

    ./view_data.py

Train models:

    ./train_model.py shape
    ./train_model.py patches
    ./train_model.py detector

View models:

    ./view_model.py shape
    ./view_model.py patches
